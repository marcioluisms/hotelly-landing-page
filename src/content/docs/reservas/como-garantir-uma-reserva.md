# Como garantir uma reserva manualmente

> **Objetivo:** Ao final deste guia, você saberá confirmar uma reserva manualmente mesmo sem o pagamento mínimo atingido — útil para hóspedes recorrentes, grupos ou acordos especiais.

## Quando usar

A confirmação automática acontece quando o hóspede paga o percentual mínimo configurado. Porém, em situações especiais, você pode querer garantir a reserva sem esperar o pagamento:

- Hóspede de confiança que sempre paga no check-in
- Reserva de grupo com pagamento combinado para depois
- Acordo comercial especial

## Passo a passo

1. No menu lateral, clique em **Painel** > **Reservas**.
2. Encontre a reserva com status **"Ag. Pagamento"**.
3. Clique no botão **"Garantir reserva"** na coluna de ações da lista de reservas.
4. No dialog **"Garantir Reserva"**, preencha a **justificativa** (campo obrigatório). Exemplo: "Hóspede recorrente, pagamento no check-in".
5. Clique em **"Garantir reserva"**.

## O que acontece depois

- O status muda de **"Ag. Pagamento"** para **"Confirmada"**, independente do saldo.
- A justificativa fica registrada na **Trilha de Auditoria** para controle e transparência.
- O quarto é garantido no **Mapa de Quartos** para o período.

## Se algo der errado

| Problema | Solução |
|---|---|
| Botão "Garantir reserva" não aparece | O botão só aparece para reservas com status "Ag. Pagamento". Se a reserva já está "Confirmada", não é necessário garantir. |
| A reserva foi garantida mas o hóspede não veio | Consulte as políticas de no-show em **Configurações** > **Propriedade** > **Regras de Reserva** para aplicar a multa adequada. |
