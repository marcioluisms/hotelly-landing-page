# Como gerenciar a limpeza de quartos

> **Objetivo:** Ao final deste guia, você saberá usar a tela de Limpeza para visualizar o status de cada quarto e atualizar o progresso da equipe de governança.

## Passo a passo

1. No menu lateral, clique em **Operação** > **Limpeza**.
2. A tela mostra todos os quartos ativos, agrupados por tipo de quarto.
3. Cada quarto exibe botões coloridos de status:

| Botão | Cor | Significado |
|---|---|---|
| **Sujo** | Vermelho | Quarto precisa de limpeza |
| **Limpando** | Azul | Equipe está limpando |
| **Limpo** | Verde | Quarto pronto para uso |

4. Para atualizar o status, basta clicar no botão correspondente:
   - Quarto sujo → clique em **"Limpando"** quando a camareira iniciar
   - Quarto limpando → clique em **"Limpo"** quando terminar
5. Cada quarto também tem um **ícone de taça** (frigobar) para registrar reposição.

## Quando os quartos ficam sujos automaticamente

- Após o **check-out** de um hóspede, o quarto muda automaticamente para **"Sujo"**
- Não é necessário marcar manualmente — o sistema faz isso

## O que acontece depois

- O **Mapa de Quartos** e a página **Início** refletem o status atualizado em tempo real.
- Se um quarto está "Sujo" ou "Limpando", o check-in fica bloqueado (mensagem: "Quarto ainda não liberado pela governança.").
- Quando marcado como "Limpo", o check-in é liberado.

## Se algo der errado

| Problema | Solução |
|---|---|
| "Nenhum quarto ativo cadastrado." | Cadastre quartos em **Configurações** > **Quartos** e marque-os como ativos. |
| Check-in bloqueado mesmo após limpeza | Verifique se o quarto foi marcado como "Limpo" na tela de Limpeza. O status precisa ser "Limpo" para liberar o check-in. |
