# Entendendo a precificação no Hotelly

> **Em uma frase:** O Hotelly oferece dois níveis de precificação — tarifas manuais no calendário (você define cada preço) e preço dinâmico inteligente (o motor sugere ajustes baseados na ocupação) — e você escolhe quanto controle quer ter.

## Por que isso importa para você

Preço errado é dinheiro perdido: cobrar pouco em alta temporada deixa receita na mesa; cobrar muito em baixa temporada afasta hóspedes. O Hotelly dá as ferramentas para acertar: o calendário para controle manual e o motor de preço dinâmico para ajustes inteligentes.

## Como funciona no Hotelly

### Nível 1: Tarifas manuais (Configurações > Tarifas)

Você define o preço de cada tipo de quarto para cada data no calendário. Isso inclui: tarifa base, adulto extra, desconto single e preços por faixa de idade de criança. É possível editar dia a dia, em lote ou usando modelos semanais reutilizáveis.

Quando uma reserva é criada, o sistema calcula o valor total automaticamente com base na tarifa da data × número de noites.

### Nível 2: Preço Dinâmico (Receita > Preço Dinâmico)

O motor analisa a ocupação prevista para os próximos 120 dias e sugere ajustes de preço. Você configura por tipo de quarto: agressividade (conservador, padrão, agressivo), preço piso e preço teto. O motor nunca aplica preços automaticamente — ele sugere, e você aceita ou ignora.

O Calendário de Picos e Feriados permite cadastrar eventos especiais com fatores de ajuste (ex: Carnaval +30%), que o motor incorpora nas sugestões.

### Como os dois se complementam

O calendário de tarifas é a base. O preço dinâmico ajusta essa base para cima ou para baixo conforme a demanda. Se você não ativar o preço dinâmico, os preços do calendário são usados como estão.
