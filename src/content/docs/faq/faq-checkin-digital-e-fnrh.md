# Perguntas frequentes sobre check-in digital e FNRH

## O hóspede precisa instalar algum aplicativo?

Não. O check-in digital é um formulário web responsivo. O hóspede abre o link no navegador do celular, preenche e pronto. Funciona em qualquer smartphone.

## O que é a FNRH?

FNRH significa Ficha Nacional de Registro de Hóspedes. É uma obrigação legal do governo brasileiro: todo meio de hospedagem precisa enviar os dados dos hóspedes à SETUR (Secretaria de Turismo) via sistema do Serpro. O Hotelly faz isso automaticamente.

## Eu pago alguma coisa pela FNRH?

Não. A API do Serpro FNRH Digital V2 é gratuita para meios de hospedagem.

## E se o hóspede não preencher o check-in digital?

Você pode coletar os dados presencialmente e registrá-los no sistema. O Hotelly envia a FNRH da mesma forma. O check-in digital é uma conveniência, não uma obrigação para o hóspede.

## O que acontece se os dados do check-in digital divergem da reserva?

O sistema detecta a divergência e exibe um alerta para a recepção. Exemplo: a reserva foi feita no nome de "Maria Silva" mas o check-in digital foi preenchido por "João Silva" (marido que vai primeiro). A recepção resolve a divergência pelo painel.

## O check-in digital substitui o check-in presencial?

Parcialmente. O hóspede preenche os dados antes de chegar, mas o check-in propriamente dito (mudar status para "Hospedado") ainda é feito pela recepção quando o hóspede chega. A vantagem é que os dados já estão no sistema e a FNRH já foi enviada.

## Hóspede estrangeiro pode fazer check-in digital?

Sim. O formulário aceita diferentes tipos de documento: CPF, RG, Passaporte, CNH e RNE (Registro Nacional de Estrangeiros). O campo se adapta ao tipo selecionado.
