# Perguntas frequentes sobre precificação

## O preço dinâmico muda meus preços automaticamente?

Não. O motor de preço dinâmico apenas **sugere** preços. Você vê a sugestão e decide: aceitar (clicando "Aplicar preço") ou ignorar. Nenhum preço é alterado sem sua confirmação.

## O que é "agressividade" do motor de precificação?

É o quanto o motor varia os preços em relação à tarifa base:

- **Conservador:** variações menores (metade do desvio normal). Bom para quem prefere estabilidade.
- **Padrão:** variações normais conforme a demanda.
- **Agressivo:** variações maiores (1,5× o desvio normal). Maximiza receita em alta demanda, mas pode afastar hóspedes em baixa.

## O que são "piso" e "teto"?

São limites de segurança. O **piso** é o preço mínimo — o motor nunca sugere abaixo dele. O **teto** é o máximo — o motor nunca sugere acima. Se você deixar o teto vazio, não há limite superior.

## Preciso usar o preço dinâmico?

Não. Se você prefere definir preços manualmente no calendário de tarifas, funciona perfeitamente. O preço dinâmico é uma ferramenta opcional para quem quer receber sugestões de otimização baseadas na ocupação e no seu calendário de eventos.

## Como funciona o preço de crianças?

As faixas de idade são configuradas em **Configurações** > **Propriedade** > **Crianças** (ex: 0-5 anos, 6-12 anos, 13-17 anos). O preço de cada faixa é definido por data no calendário de tarifas, ao lado da tarifa base. Crianças fora das faixas configuradas são tratadas como adultos.

## Posso cobrar preços diferentes no fim de semana?

Sim. Use o **Modelo semanal** no calendário de tarifas: defina preços diferentes para sexta/sábado/domingo e salve como modelo reutilizável. Ou edite os dias manualmente.

## O que acontece se eu não definir tarifa para uma data?

O sistema não permite criar reservas para datas sem tarifa. A mensagem "Tarifa indisponível" aparece. Defina tarifas para todos os dias em que aceita reservas.
