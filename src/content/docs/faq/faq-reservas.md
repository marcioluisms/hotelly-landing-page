# Perguntas frequentes sobre reservas

## Posso criar uma reserva sem cobrar nada do hóspede?

Sim. Ao criar a reserva, ela fica com status "Ag. Pagamento". Você pode usar o botão **"Garantir reserva"** para confirmá-la sem esperar pagamento. A justificativa fica registrada na Trilha de Auditoria.

## Como sei se o hóspede pagou?

Na lista de reservas, a coluna **Saldo** mostra o valor em cores: **vermelho** se há saldo devedor, **verde** se o hóspede tem crédito, e **traço cinza** se está zerado. Para detalhes, abra a reserva e consulte o card **"Financeiro"**.

## Consigo alterar o valor total de uma reserva?

O valor é calculado automaticamente pela tarifa. Para ajustar, use **"Alterar datas"** (que permite informar um ajuste em R$) ou adicione/remova extras. Não há campo para editar o valor total diretamente — isso garante rastreabilidade.

## O que acontece quando dois atendentes editam a mesma reserva?

O sistema detecta o conflito e mostra: "Reserva foi alterada por outro atendente. Recarregue a página." Recarregue para ver os dados atualizados e tente novamente.

## Posso reabrir uma reserva cancelada?

Não. Uma vez cancelada, a reserva não pode ser revertida. Crie uma nova reserva para o mesmo hóspede e período.

## Posso fazer check-in parcial (só alguns hóspedes)?

Não. O check-in é da reserva inteira. Se um hóspede chega antes dos outros, faça o check-in normalmente — os demais são registrados como parte da mesma reserva.

## Como funciona a reserva que vem pelo WhatsApp?

O Concierge IA conversa com o hóspede, verifica disponibilidade e envia um link de pagamento. Quando o hóspede paga, a reserva é criada automaticamente no sistema com a fonte registrada como WhatsApp. Aparece na lista de reservas como qualquer outra.

## O Motor de Reservas (site) e o WhatsApp podem vender o mesmo quarto?

Não. O inventário é unificado. Quando um quarto é reservado por qualquer canal (manual, WhatsApp ou Motor de Reservas), a disponibilidade é atualizada em tempo real para todos os canais. Overbooking é impossível por design.
