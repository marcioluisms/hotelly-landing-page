# Perguntas frequentes sobre o Copilot

## Qual a diferença entre o Copilot e o Concierge IA?

O **Copilot** é o assistente interno para a equipe — fica dentro do dashboard e ajuda com dúvidas sobre o sistema e dados da operação. O **Concierge IA** é o assistente externo para hóspedes — funciona pelo WhatsApp e ajuda com informações da hospedagem e reservas.

## O Copilot pode executar ações no sistema?

Sim. O Copilot utiliza "function calling" para executar ações diretamente: consultar dados de reservas, verificar disponibilidade, resumir o dia, entre outras. As ações disponíveis dependem do perfil de acesso do usuário — uma camareira não pode consultar dados financeiros pelo Copilot, por exemplo.

## O Copilot pode inventar dados?

O Copilot consulta dados reais do sistema em tempo real. Para perguntas sobre "como fazer", ele usa a base de conhecimento da documentação. Se não encontrar a resposta, ele informa que não sabe.

## Qualquer perfil pode usar o Copilot?

Sim, todos os perfis têm acesso ao Copilot. Mas as informações e ações disponíveis respeitam as permissões do perfil: um Visualizador pode consultar mas não pode executar ações, uma Governança só vê informações de limpeza.

## O Copilot funciona em português?

Sim. Ele opera em português e converte automaticamente valores (centavos para R$, datas no formato brasileiro).
