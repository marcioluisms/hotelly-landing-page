# Obrigações legais — FNRH e SETUR

## O que é a FNRH

A Ficha Nacional de Registro de Hóspedes (FNRH) é uma obrigação legal do governo brasileiro. Todo meio de hospedagem é obrigado a registrar os dados dos hóspedes e enviar essas informações à SETUR (Secretaria de Turismo) através do sistema do Serpro.

A FNRH é regulamentada pela Lei 11.771/2008 (Lei Geral do Turismo) e pelo Decreto 7.381/2010.

## O que o Hotelly faz automaticamente

O Hotelly envia a FNRH ao Serpro de forma automática, usando a API FNRH Digital V2. Quando os dados do hóspede são preenchidos — seja pelo check-in digital (formulário online) ou pelo cadastro manual na recepção — o sistema transmite as informações obrigatórias sem que a equipe precise acessar nenhum portal externo.

## Dados obrigatórios da FNRH

Para cada hóspede, são necessários: nome completo, documento de identidade (CPF, RG, Passaporte, CNH ou RNE), nacionalidade, data de nascimento, e dados da estadia (check-in, check-out, quarto). Esses campos são coletados automaticamente pelo formulário de check-in digital ou pelo cadastro manual.

## Custo

A API do Serpro FNRH Digital V2 é gratuita para meios de hospedagem. Não há custo adicional do Hotelly para essa funcionalidade.

## E se o hóspede não preencher o check-in digital?

O envio da FNRH acontece com base nos dados que estão no sistema, independentemente de como foram inseridos. Se o hóspede não preencheu o check-in digital, a recepção pode registrar os dados manualmente pelo painel. O Hotelly envia a FNRH da mesma forma.

## Hóspedes estrangeiros

O formulário aceita diferentes tipos de documento: CPF, RG, Passaporte, CNH e RNE (Registro Nacional de Estrangeiros). O campo se adapta ao tipo selecionado.

## Responsabilidade do hoteleiro

A obrigação legal de enviar a FNRH é do meio de hospedagem. O Hotelly automatiza o processo, mas é responsabilidade do hoteleiro garantir que os dados dos hóspedes estejam corretos e completos no sistema antes do check-out.
