# Guia rápido — Financeiro

Este guia resume o que você pode fazer no Hotelly como Financeiro (Finance) e como usar o sistema para acompanhar receita e pagamentos.

## O que você pode fazer

- Acessar relatórios financeiros: Receita (caixa e competência), Ocupação, ADR, RevPAR
- Exportar relatórios em CSV
- Consultar a Trilha de Auditoria (cancelamentos, reembolsos, alterações de valor)
- Visualizar o folio de reservas (pagamentos recebidos, saldo pendente)
- Acompanhar o status de pagamentos (confirmados, pendentes, reembolsados)
- Usar o Copilot para consultas financeiras rápidas

## O que você não pode fazer

- Criar, editar ou cancelar reservas
- Fazer check-in ou check-out
- Alterar configurações da propriedade (tarifas, quartos, políticas)
- Convidar ou remover membros da equipe
- Acessar o painel de Governança
- Configurar o Concierge IA ou Motor de Reservas

## Rotina sugerida

**Diária:** Consulte o relatório de Receita no regime **Caixa** para ver quanto entrou no dia. Verifique se há pagamentos pendentes em reservas ativas.

**Semanal:** Exporte o relatório de Receita da semana em CSV para conciliação. Consulte a Trilha de Auditoria para verificar reembolsos e cancelamentos que impactam o caixa.

**Mensal:** Compare os relatórios de **Caixa** e **Competência** para entender a diferença entre o que entrou no mês e o que efetivamente se refere a estadias daquele período. Acompanhe ADR e RevPAR para avaliar a performance tarifária. Exporte os dados consolidados para sua contabilidade.

## Dica

Use o Copilot para consultas rápidas como "qual a receita desta semana?" ou "quantos reembolsos tivemos neste mês?". Ele puxa os dados em tempo real sem precisar navegar entre telas.
