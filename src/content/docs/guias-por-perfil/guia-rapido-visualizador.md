# Guia rápido — Visualizador

Este guia resume o que você pode fazer no Hotelly como Visualizador (Viewer) — o perfil de acesso somente leitura.

## O que você pode fazer

- Visualizar reservas e seus detalhes (datas, quarto, status)
- Consultar o Mapa de Quartos e a ocupação geral
- Ver o Dashboard com indicadores do dia
- Usar o Copilot para tirar dúvidas sobre o sistema

## O que você não pode fazer

- Criar, editar ou cancelar reservas
- Fazer check-in ou check-out
- Registrar pagamentos
- Acessar relatórios financeiros detalhados
- Alterar qualquer configuração
- Convidar ou remover membros da equipe
- Interagir com o painel de Governança

## Para quem é esse perfil

O perfil Visualizador é ideal para sócios que querem acompanhar a operação sem interferir, consultores externos que precisam de visibilidade temporária, ou qualquer pessoa que precisa ver o andamento da hospedagem sem executar ações.

## Dica

Mesmo como Visualizador, você pode usar o Copilot para perguntas como "quantas reservas temos para este fim de semana?" ou "qual a taxa de ocupação hoje?". O Copilot respeita as permissões do seu perfil — mostra apenas informações que você tem autorização para ver.
