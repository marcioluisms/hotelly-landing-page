# Como configurar seu estabelecimento

Este é o primeiro passo. Você vai cadastrar as informações da sua hospedagem — endereço, contato e dados fiscais. Leva menos de 10 minutos.

## O que você vai precisar

- Endereço completo da hospedagem (com CEP)
- Telefone e e-mail de contato
- CNPJ ou CPF do responsável (pode preencher depois)

## Passo a passo

### 1. Acesse as configurações

No menu esquerdo, clique em **Configurações**. Na aba **Propriedade**, localize a seção **Estabelecimento**.

### 2. Preencha o Endereço

O primeiro grupo de campos é o de endereço:

- **CEP** — formato 00000-000. Ao digitar, os demais campos de endereço podem ser preenchidos automaticamente.
- **Logradouro** — rua, avenida ou estrada.
- **Número** — número do imóvel.
- **Complemento** — bloco, sala, andar (se houver).
- **Bairro** — bairro da hospedagem.
- **Cidade** — nome da cidade.
- **UF** — selecione o estado no dropdown (27 opções, de AC a TO).

### 3. Preencha o Contato

O segundo grupo traz os dados de comunicação:

- **Telefone** — formato (00) 00000-0000. A máscara é aplicada automaticamente ao digitar.
- **E-mail** — endereço de e-mail da hospedagem (validado pelo sistema).

### 4. Preencha os Dados Fiscais

O terceiro grupo é para documentação legal:

- **Tipo de documento** — alterne entre CNPJ e CPF. O campo muda a máscara automaticamente:
  - **CNPJ** — formato 00.000.000/0000-00
  - **CPF** — formato 000.000.000-00
- **Inscrição Estadual** — se tiver. Campo livre, sem máscara.

**Nota:** Os dados fiscais são opcionais no início. Recomendamos preencher antes de ativar o check-in digital, pois o CNPJ/CPF é necessário para os envios da FNRH ao governo.

### 5. Clique em **Salvar**

Pronto. As informações ficam registradas e você pode editá-las a qualquer momento.

## O que acontece depois

- O nome da hospedagem aparece nos e-mails enviados aos hóspedes.
- O endereço é usado pelo Concierge IA para responder perguntas de localização.
- O telefone pode ser incluído em respostas automáticas do Concierge.
- Os dados fiscais são usados para gerar documentos legais (FNRH, recibos).

## Se algo der errado

**"Não consigo encontrar a seção Estabelecimento"**
Ela fica dentro de **Configurações** > **Propriedade**. Se o menu estiver recolhido, clique no ícone de três linhas (≡) para expandir.

**"O sistema não aceita meu CEP / telefone / CNPJ"**
Verifique se o formato está correto. O sistema aplica máscara automática — basta digitar os números que a formatação aparece. Se o problema persistir, abra um ticket de suporte.

**"Posso mudar os dados depois?"**
Sim. Volte a **Configurações** > **Propriedade** > **Estabelecimento** e edite qualquer campo a qualquer momento.

**"Preciso preencher o CNPJ agora?"**
Não é obrigatório no início. Mas preencha antes de ativar o check-in digital — o sistema precisa do documento fiscal para enviar a FNRH.
