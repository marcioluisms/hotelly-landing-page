# Como cadastrar tipos de quarto

Um "tipo de quarto" é a categoria — Quarto Simples, Quarto Duplo, Suite, etc. Você cria o tipo uma vez e depois vincula quantos quartos físicos quiser a ele. Leva 15 minutos.

## O que você vai precisar

- Lista dos tipos de quarto que sua hospedagem oferece
- Capacidade máxima de cada tipo (quantas pessoas)
- Descrição curta (opcional, mas ajuda)

## Entenda a diferença

**Tipo de quarto** = categoria (ex: "Duplo com Ar")
**Quarto** = o quarto real no corredor (ex: "Quarto 101")

Um tipo pode estar em vários quartos. Exemplo: 5 quartos diferentes, todos do tipo "Duplo com Ar".

## Passo a passo

### 1. Acesse o cadastro de tipos
No dashboard, vá em **Configurações** > **Quartos** > aba **"Tipos de Quarto"** (ou clique **+ Novo Tipo de Quarto**).

### 2. Clique em **+ Novo Tipo de Quarto**
Um formulário aparecerá.

### 3. Preencha os dados

**Nome do Tipo**
Algo claro e descritivo. Exemplos:
- "Quarto Simples"
- "Duplo com Ar"
- "Suite Master"
- "Dormitório 4 camas"

Placeholder: "Ex: Suíte Master"

**Descrição (opcional)**
Uma frase sobre o quarto. Exemplo: "Quarto amplo com ar-condicionado, wi-fi e frigobar". Você vai usar isso depois na descrição para hóspedes.

**Capacidade máxima (hóspedes)**
Quantas pessoas o quarto acomoda no máximo? Exemplo: 2 pessoas. Campo numérico de 1 a 20.

### 4. Clique em **Salvar**

O tipo fica registrado. Agora você pode criar quantos quartos quiser deste tipo.

### 5. Repita para cada tipo
Se sua hospedagem tem 3 tipos diferentes (Simples, Duplo, Suite), repita os passos 2-4 para cada um.

## O que acontece depois

- Estes tipos aparecem quando você criar quartos
- Hóspedes verão estes tipos no site / WhatsApp
- O sistema usa o tipo para agrupar relatórios (quanto cada tipo rendeu)
- A precificação dinâmica ajusta preço por tipo (não por quarto individual)

## Para definir preços

Os preços são configurados **em Configurações > Tarifas** (página separada com calendário), não aqui. Veja "Como definir preços e tarifas" para configurar as tarifas por data e tipo.

## Se algo der errado

**"Posso mudar o nome depois?"**
Sim. Clique em **Editar** no tipo existente. Mas cuidado: se já tem hóspedes reservados, a mudança pode ficar confusa. Melhor acertar agora.

**"Falta um tipo. Como adiciono?"**
Repita o passo 2-4. Você pode criar quantos tipos precisar.

**"Quero deletar um tipo que não uso mais."**
Clique em **Remover**. O sistema bloqueia se houver quartos ativos vinculados ou reservas confirmadas/em andamento. Desative os quartos primeiro.
